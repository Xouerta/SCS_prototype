# SCS_prototype
## 本项目拟设计并实现一套智能 Web 威胁态势感知与管理系统，旨在利用机器学习技术解决传统网络安全防御中规则匹配僵化的问题，并结合现代化 Web 工程技术提升系统的可维护性与演示交互效果。

本系统采用前后端分离及多语言混合架构
- 前端基于 Vue.js 3 生态构建，引入 qiankun 微前端框架将系统拆分为“态势感知可视化大屏”与“后台管理系统”两个独立子应用，既保证了炫酷数据展示与常规业务管理的解耦，又体现了现代前端的大型应用架构思维；
- 后端采用 Node.js (Koa2/NestJS) 与 Python (FastAPI) 的双服务分布式设计，其中 Node.js 作为主业务网关，负责处理用户鉴权、数据接入与路由分发，鉴权模块采用成熟的账号密码结合动态图形验证码（SVG Captcha）机制以确保安全性并简化开发；
- Python 服务则专注于计算密集型任务，加载基于 轻量级机器学习模型（包括恶意 URL 识别、SQL 注入检测， XSS 跨站脚本检测），实现对恶意流量的高效推理。
- 为了解决高并发流量下的系统阻塞问题并增加技术亮点，系统引入 Redis 消息队列实现异步通信，Node.js 接收到的模拟攻击流量（由 Python 脚本生成的背景噪音与突发攻击）先写入队列，再由 Python 服务消费处理，处理结果最终通过 WebSocket 协议实时推送至前端大屏，利用 ECharts 绘制动态攻击源地图与流量波形图。

该设计不仅完整演示了从数据采集、队列缓冲、AI 智能分析到实时可视化的全链路流程，还通过模块化设计大幅降低了模型训练与系统部署的硬件门槛
